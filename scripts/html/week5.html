WEEK 5 - Analysis of The World-Wide Web

Background of The World-Wide Web The World-Wide Web (W-3, or simply web) is a project designed to allow information exchange between people. The Web is based on the use of hypermedia, enabling the user to link to any other information, no matter where it is on the Internet. The people of the Web share a common technical ideology, and we are striving towards common goals. The popularity of the Internet has allowed the Web to flourish, and it is now one of the most important aspects of the Internet. The Web has its own project space, an mail service, and member home pages. This project space is useful in maintaining Web documents, as things are always changing and it's difficult to program HTML without a browser. The Web now has many browsers: Line mode, Viola, 0.18, Arena, Mosaic and Samba, to name a few. When the Web first began, it was difficult to find out what was happening with it, and Mosaic was not a public domain browser. This has changed. Mosaic is currently available for most window systems, and the most recent versions may be obtained from NCSA via anonymous FTP.

Transforming Information Access An even more profound effect is the potential for multimedia documents to be created and linked together. A multimedia document is one which is generated by combining information from different media. For example, a hypertext document containing tagged text and images is a multimedia document. At present, most hypertext documents are simple text with optional inline images. The reason for this is there is as yet no standard way to generate and view more complex multimedia documents. Often the only way to view a complex multimedia document is to use a computer and communication system of the same type as was used to create the document. This reduces document portability. However, with the rapid progress being made in this field, it is likely that soon it will be possible to create and view multimedia documents of all types using nothing more than a generic web browser and an Internet connection. This will set a new global standard for document portability. Further to transforming methods of communication, the web is beginning to revolutionize the dissemination of information. It is doing so through merging previously separable media such as text and graphics, and providing an interface to a vastly increased array of document types. The ease with which documents can be accessed over the web far surpasses other still developing information superhighways such as CD-ROM. With systems such as the URL-based WWW Gopher, it has become possible to access and browse a great variety of information without knowing or needing to know the physical location of the information. This is of great significance because it means the end of queuing in libraries and learning the location of information as a separate task to learning the information itself.

Future Prospects and Implications Backwards compatibility will always be a major issue. If future internet technologies such as distributed computing, distributed multimedia, and synchronised document editing are to embrace the web, they will each need to access and manipulate web objects such as documents and forms. They should reuse and extend existing web technology, rather than bypassing it. It is clearly stated that the future of document technology will enable people to move far beyond the paper-like resources which form the current web. This too must be done in a way which does not bypass the existing medium. Our hope is that the principle of universal access will be maintained as the web migrates to new technologies and media types. The technologies of the web infrastructure will continue to evolve. New hardware, for example, will probably incorporate firewalls, routers and network connectivity more commonly than is the case for present PCs. The web's security and internationalism may well be threatened if future operating systems and networked file systems come to assume complex user identities and proprietary metadata. The web is now well established. It may be interesting to compare this dynamic age of the web with a possible later stationary or even declining age, and to try to foresee the changes. Although no aspects of the future are cast in stone, this section attempts to highlight some emerging technologies and trends. Only those with a clear potential to influence the web positively or negatively are discussed.
